# Image Gallery with Horizontal Scrolling

This project demonstrates how to create a simple horizontal-scrolling image gallery using HTML, CSS, and JavaScript. Users can navigate through the gallery by clicking on "Next" and "Back" buttons or by using the mouse wheel.

## Prerequisites

Ensure you have the following files in your project:

- `index.html`: HTML file for the gallery layout.
- `style.css`: CSS file for styling.
- `script.js`: JavaScript file for gallery functionality.
- `images/`: Directory containing your gallery images.
